==== Sujet ====

Création d'un site internet pour l'association de musique Musik'Eole.

==== Fonctionnalités principales ====

- Pages modifiables par l'administrateur (Pages de présentations, agenda, ...)
- Galeries photos
- Système de membres
- Forum
- Dépot d'annonces
- Boutique

==== Technologies ====

- HTML5/CSS3
- PHP5
- Framework CSS Bootstrap
